# PNPC Pocket Service Desk

A comprehensive WordPress service desk plugin for managing customer support tickets with WooCommerce integration.

## Installation

The plugin is located in the `pnpc-pocket-service-desk` directory. To install:

1. Copy the `pnpc-pocket-service-desk` directory to your WordPress installation's `wp-content/plugins/` directory
2. Activate the plugin through the WordPress admin 'Plugins' menu
3. Configure the plugin settings from the Service Desk menu in WordPress admin

## Features

See the plugin's README.md file in the `pnpc-pocket-service-desk` directory for complete documentation.

## Quick Start

After installation, use these shortcodes on your WordPress pages:

- `[pnpc_service_desk]` - Main service desk dashboard
- `[pnpc_create_ticket]` - Ticket creation form
- `[pnpc_my_tickets]` - List of user's tickets
- `[pnpc_profile_settings]` - Profile settings with image upload

For more information, see the full documentation in `pnpc-pocket-service-desk/README.md`.
